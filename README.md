# Upload-File-in-NodeJS-Using-Multer
This is a simple application that  uploads a file using multer module in Node JS. Please note that before running this program there should a folder named 'uploads' in your project directory.

Get the required node modules :
npm install multer
npm install express

1) Run the project as : 
node uploader.js

2) Navigate to localhost:3000 to see the homepage as seen below: 

![alt tag](https://github.com/sagardafle/Upload-File-in-NodeJS-Using-Multer/blob/master/uploads/Screenshot%20from%202016-06-05%2002-51-34.png)

3) Browse to the file yu wish to upload ans select the file.
![alt tag](https://github.com/sagardafle/Upload-File-in-NodeJS-Using-Multer/blob/master/uploads/Screenshot%20from%202016-06-05%2002-51-52.png)

4) After the success message, navigate to the uploads folder to check if the file is uploaded. 
The selected file is the file that I had chose while uploading.
![alt tag](https://github.com/sagardafle/Upload-File-in-NodeJS-Using-Multer/blob/master/uploads/Screenshot%20from%202016-06-05%2002-52-12.png)
